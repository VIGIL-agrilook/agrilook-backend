# 데이터 파일 설명

이 폴더에는 비료 처방 시스템에서 사용하는 데이터 파일들이 포함되어 있습니다.

## 파일 목록

- `밑거름.json`: 기본 비료(밑거름) 제품 데이터 (137개 제품)
- `웃거름.json`: 추가 비료(웃거름) 제품 데이터 (35개 제품)

## 데이터 구조

각 JSON 파일은 다음과 같은 구조를 가집니다:

```json
[
  {
    "비료종류": "제품명",
    "질소": "N 함량(%)",
    "인산": "P 함량(%)", 
    "칼리": "K 함량(%)",
    "포장단위": "포장 단위(kg)"
  }
]
```

## 데이터 출처

- 농촌진흥청 토양환경정보시스템
- 국내 비료 제조업체 공식 제품 정보
